Videogames are everywhere and they are a lot of fun but do you know how they work?

In order to hack videogames you will need to know how they work. To get you up to speed, we will be reviewing some important gaming concepts. Ready?, let’s go!

Let's talk about memory first, specifically about computer memory. So what's that? Well memory is any physical device capable of storing information temporarily

One of the most common devices for storing working data is the RAM, or Random Access Memory

But why is it called Random?, because data can be written or read in any order. This means that you need a way to locate data within the memory

Let me introduce you to, MEMORY ADDRESSES!, think of them as your and your friend's home addresses!. They are used to locate data (homes) within the memory (your neighborhood)

In other words, a memory address is a reference to a specific memory location

But if data is stored in memory, where are memory addresses stored?. They are stored in what is known as a pointer.Think of a pointer as a piece of paper in which you write your home address, which is then used to locate your home.

When you obtain the value stored at a location specified by the pointer, you are "dereferencing the pointer"

Now that you know what memory is let me tell you that a game is like any other computer program. When you run a game, the operating system loads the game code into memory before running it

Aside from the game code, memory can hold data like the amount of health and money of your character, these are also known as VARIABLES

If you know the location of these variables in memory, you will be able to modify them!, Imagine having infinite health!

If you have been paying attention so far, you already know that these locations are also known as Memory Addresses , but how do you get them?

The answer is CHEAT ENGINE!

Cheat Engine is one of the most popular open source applications used to hack games. With it you can scan the memory, search for specific values in the memory and more!

Cheat Engine uses a trial an error approach for finding the addresses of the desired game values. You need to progressively filter the addresses until you find the correct ones
